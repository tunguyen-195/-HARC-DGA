# HARC-DGA

**HARC-DGA: A Risk-Coverage Benchmark for Open-World DGA Family Attribution in Network-Security Triage**

This repository is the public landing page for the HARC-DGA research artifact.
The study evaluates open-world DGA family attribution with risk-coverage
reporting and unknown-family false-family attribution (UFFA).
